ВАЖНАЯ ИНФОРМАЦИЯ

1. Программа предназначена для сортировки картинок по разрешению, ничего более.

2. Программа "понимает" около 15 форматов изображений, любые неподдерживаемые файлы (даже вовсе не картинки) будут помещены в папку "неподдерживаемые".

3. Программа НЕ "смотрит" на расширение файла, она анализирует содержимое, и даже может исправить некорректное расширение если такое будет обнаружено.

4. Если в конечной папке существует файл с таким же именем, новый файл будет переименован (к его имени будет добавлена цифра), перезапись файлов невозможна.

5. Список поддерживаемых форматов может меняться в зависимости от платформы.
