IMPORTANT

1. This program does the sorting images by resolution, nothing more.

2. This program supports about 15 images formats, any unsupported files (even not images) will placed to "unsupported" folder.

3. This program relies on file content, not extension, and even can fix incorrect file extension when it detected.

4. If file with same name already exists in target directory, new file will be renamed (number will appended to its name), no overwrites is guaranted.

5. List of supported formats may vary depending on platform.
